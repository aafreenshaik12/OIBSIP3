# Weather App

## Description

The Weather App is a Python-based desktop application developed using the Tkinter library. It allows users to search for the current weather conditions of any city by entering the city name. The application retrieves real-time weather information using a weather API and displays details such as temperature, humidity, weather condition, and wind speed through a simple graphical user interface.

## Features

* User-friendly graphical interface
* Search weather by city name
* Displays current temperature
* Shows humidity and wind speed
* Displays weather condition
* Real-time weather updates using an API
* Input validation and error handling

## Technologies Used

* Python 3
* Tkinter
* Requests
* Weather API

## Installation

1. Clone the repository:

   ```bash
   git clone <repository-link>
   ```

2. Navigate to the project directory:

   ```bash
   cd Weather-App
   ```

3. Install the required package:

   ```bash
   pip install requests
   ```

4. Add your Weather API key to the project.

5. Run the application:

   ```bash
   python weather_app.py
   ```

## How It Works

1. Launch the application.
2. Enter the name of a city.
3. Click the **Search** button.
4. The application fetches real-time weather data from the API.
5. Current weather details are displayed on the screen.

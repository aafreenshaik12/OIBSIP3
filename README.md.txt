# Basic Weather App

## Objective

The Basic Weather App is a Python application that fetches and displays real-time weather information for any city using the OpenWeatherMap API.

## Features

* Search weather by city name
* Displays temperature in Celsius and Fahrenheit
* Shows humidity level
* Shows weather condition/description
* Displays wind speed
* Handles invalid city names
* Handles network and API errors

## Technologies Used

* Python
* Requests Library
* OpenWeatherMap API

## How to Run

1. Install the required package:

```bash
pip install requests
```

2. Add your OpenWeatherMap API key in the code.

3. Run the application:

```bash
python weather_app.py
```
